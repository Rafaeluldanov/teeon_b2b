# 06. Header, Footer и навигация

## Header

Файл: `components/Header/Header.tsx`  
Стили: `components/Header/Header.module.css`

### Структура Header

Header фиксированный (sticky), присутствует на всех страницах через `app/layout.tsx`. Состоит из:
- Логотип TEEON (ссылка на главную `/`)
- Основная навигация (desktop)
- Кнопка «Проконсультироваться» (CTA)
- Мобильное меню (бургер)

### Пункты навигации

| Пункт | Тип | Ссылка/Поведение |
|---|---|---|
| **Каталог** | Dropdown | Открывает список 8 категорий |
| **Нанесение** | Dropdown | Открывает список 9 методов брендирования |
| **Сувениры** | Ссылка | `/suvenirnaya-produkciya/` |
| **Портфолио** | Ссылка | `/portfolio/` |
| **О компании** | Ссылка | `/about/` |
| **Контакты** | Ссылка | `/contacts/` |
| **Проконсультироваться** | Кнопка-ссылка | `href="#request"` (только на главной и /contacts/) |

### Dropdown «Каталог»

Содержит ссылки на все 8 категорий:
- Футболки → `/catalog/futbolki/`
- Худи → `/catalog/hudi/`
- Свитшоты → `/catalog/svitshoty/`
- Лонгсливы → `/catalog/longslivy/`
- Сумки → `/catalog/sumki/`
- Жилетки → `/catalog/zhiletki/`
- Куртки → `/catalog/kurtki/`
- Дождевики → `/catalog/dozhdeviki/`

Внизу dropdown: ссылка «Весь каталог» → `/catalog/`

### Dropdown «Нанесение»

Содержит ссылки на все 9 методов брендирования:
- Вышивка → `/branding/vyshivka/`
- Шевроны → `/branding/shevrony/`
- Шелкография → `/branding/shelkografiya/`
- DTF-печать → `/branding/dtf-pechat/`
- DTG-печать → `/branding/dtg-pechat/`
- Сублимация → `/branding/sublimaciya/`
- Тиснение → `/branding/tisnenie/`
- Гравировка → `/branding/gravirovka/`
- Бирки → `/branding/birki/`

Внизу dropdown: ссылка «Все способы нанесения» → `/branding/`

### Контакты в Header

Телефон в Header берётся из `lib/contacts.ts` → `contacts.phone`. Не хардкодить напрямую в компоненте.

### Активное состояние

Header использует `usePathname()` из Next.js для определения текущего маршрута и подсветки активного пункта навигации. Чтобы не сломать активное состояние — не изменяйте механизм определения пути и не переименовывайте маршруты без соответствующего обновления логики в Header.

### Кнопка «Проконсультироваться»

Ссылка `href="#request"`. Скроллит к форме заявки. Работает корректно только на страницах, где есть элемент с `id="request"` — это главная (`/`) и контакты (`/contacts/`). На остальных страницах кнопка ведёт на `/#request`.

## Footer

Файл: `components/Footer/Footer.tsx`  
Стили: `components/Footer/Footer.module.css`

### Структура Footer

Footer присутствует на всех страницах. Содержит:
- Логотип и описание компании
- Навигационные колонки (Каталог, Брендирование, Компания)
- Контакты (телефон, email, Telegram)
- Нижняя строка: копирайт, ссылка на политику ПД, ссылка на реквизиты

### Контакты в Footer

Все контактные данные берутся из `lib/contacts.ts`. Для обновления — редактировать только этот файл.

## Где что менять

### Изменить телефон или email

Файл: `lib/contacts.ts`
```ts
export const contacts: ContactsData = {
  phone: '+7 (999) 000-00-00',   // ← сюда
  email: 'info@teeon.ru',         // ← сюда
  telegram: '@teeon_merch',       // ← сюда
  ...
}
```

### Добавить пункт в навигацию

1. Открыть `components/Header/Header.tsx`
2. Найти массив навигационных ссылок
3. Добавить новый объект `{ label: 'Название', href: '/url/' }`
4. Убедиться, что страница существует в `app/`

### Изменить ссылки в dropdown

Ссылки дропдаунов формируются из данных `lib/catalog.ts` и `lib/branding.ts`. Если добавляете новую категорию или метод — сначала добавьте данные в соответствующий lib-файл, затем ссылка автоматически появится в дропдауне.

### Изменить копирайт

Файл: `components/Footer/Footer.tsx` — строка с годом и названием компании.

## Мобильное меню

На мобильных устройствах Header показывает бургер-кнопку. При нажатии открывается мобильное меню со всеми пунктами навигации. Dropdown-пункты раскрываются аккордеоном.

Закрывается при:
- Клике на любую ссылку
- Клике на крестик (кнопка закрытия)
- Нажатии Escape
