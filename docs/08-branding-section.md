# 08. Раздел брендирования

## Технологии брендирования

В проекте реализовано 9 методов нанесения логотипа и персонализации:

| Slug | Название | Основное применение |
|---|---|---|
| `vyshivka` | Вышивка | Худи, поло, куртки, кепки — объёмное и долговечное нанесение |
| `shevrony` | Шевроны | Нашивки на одежду — готовый элемент с оверлоком или лазерной резкой |
| `shelkografiya` | Шелкография | Футболки, свитшоты — тиражная печать от 50 шт. |
| `dtf-pechat` | DTF-печать | Любые изделия — фотографическое качество, малые тиражи |
| `dtg-pechat` | DTG-печать | Хлопковые изделия — прямая печать с фотографическим качеством |
| `sublimaciya` | Сублимация | Синтетика — полноцветная печать, no-fade |
| `tisnenie` | Тиснение | Кожа, ПВХ, бумага — рельефный логотип без чернил |
| `gravirovka` | Гравировка | Металл, дерево — лазерная гравировка |
| `birki` | Брендированные бирки | Внутренние и навесные бирки с логотипом |

## Где находятся данные

| Файл | Содержание |
|---|---|
| `lib/branding.ts` | Основные данные методов: описание, SEO, применимые изделия, преимущества, ограничения, процесс |
| `lib/brandingSamples.ts` | Образцы работ для каждого метода (показываются на страницах брендирования) |

## Структура данных метода (lib/branding.ts)

```ts
{
  slug: 'vyshivka',
  title: 'Вышивка',
  menuTitle: 'Вышивка',           // Название в меню Header
  h1: 'Вышивка логотипа на одежде',
  shortDescription: '...',        // Для карточки на главной и в каталоге
  description: '...',             // Подробное описание
  icon: '🪡',
  tags: ['Корпоративный мерч', ...],
  seoTitle: '...',
  seoDescription: '...',
  suitableFor: ['...'],           // Для кого подходит
  products: ['Худи', 'Куртки', ...],  // На каких изделиях применяется
  benefits: ['...'],              // Преимущества
  limitations: ['...'],          // Ограничения
  process: ['...'],              // Этапы работы
  priceFactors: ['...'],         // Что влияет на стоимость
  mockupRequirements: '...',     // Требования к макету
  examples: [{ name, task }],    // Примеры применения
  relatedMethods: ['shevrony', ...],   // Связанные методы
  relatedCatalog: ['futbolki', ...],   // Связанные категории каталога
}
```

## Как добавить новую технологию брендирования

1. Добавить объект в массив `brandingMethods` в `lib/branding.ts`
2. Создать папку страницы: `app/branding/<slug>/page.tsx`
3. В файле страницы использовать стандартный паттерн:
```tsx
import { getBrandingMethod } from '@/lib/branding';
import BrandingPageContent from '@/components/BrandingPageContent/BrandingPageContent';

export default function Page() {
  const method = getBrandingMethod('new-slug')!;
  return <BrandingPageContent method={method} />;
}
```
4. Добавить slug в sitemap — он подхватится автоматически из `brandingMethods`
5. Добавить образцы в `lib/brandingSamples.ts`

## Как редактировать тексты

Все тексты методов брендирования — в `lib/branding.ts`. Найдите нужный метод по `slug` и отредактируйте нужное поле.

## Образцы работ (lib/brandingSamples.ts)

Каждый метод может иметь образцы — карточки с примерами работ. Структура образца:
```ts
{
  slug: 'vyshivka',
  samples: [
    {
      name: '3D-вышивка на кепке',
      imageSrc: '/images/branding/vyshivka/3d-cap.jpg',  // или null для заглушки
      description: '...',
      tags: ['3D', 'Кепки'],
    }
  ]
}
```

## Где хранить изображения брендирования

```
public/images/branding/<slug>/<filename>.jpg
```

Примеры:
```
public/images/branding/vyshivka/3d-cap.jpg
public/images/branding/shelkografiya/puff-print.jpg
public/images/branding/dtf-pechat/gradient-logo.jpg
```

Рекомендуемый размер: **800×600 px** или **600×400 px**.
