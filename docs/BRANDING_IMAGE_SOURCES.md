# BRANDING IMAGE SOURCES

Таблица источников и лицензий для изображений вариантов нанесения.

Синхронизируется с `public/branding-samples/sources.json` (тот же контент в машинно-читаемом виде).

## Зачем
- Подтверждение, что для каждой картинки есть право на использование на коммерческом сайте.
- Защита от претензий правообладателей.
- Возможность повторного скачивания / связи с автором, если файл потеряется.

## Допустимые лицензии
- **Unsplash License** — бесплатно для коммерческого использования, атрибуция желательна но не обязательна. <https://unsplash.com/license>
- **Pexels License** — бесплатно для коммерческого использования, атрибуция не обязательна.
- **Pixabay Content License** — бесплатно, атрибуция не обязательна.
- **Wikimedia Commons / Wikipedia** — только лицензии CC0, CC-BY, CC-BY-SA, Public Domain. CC-NC и аналоги — НЕ подходят.
- **Собственные фото TEEON** — лицензия `TEEON-Own`, автор: внутренний фотограф / съёмочная группа.
- **Купленные на стоке** — название стока + номер заказа.

## Текущее наполнение

Все 39 фотографий загружены с Unsplash 2026-05-25 в рамках первой партии иллюстраций раздела «Нанесение».
Атрибуция авторов сохранена. Многие фото — иллюстративные стоковые изображения, которые **рекомендуется заменить на собственные фотографии работ TEEON** перед публикацией продакшна. До замены они выполняют функцию мокапов для адаптивной вёрстки, SEO og:image и общего восприятия раздела.
