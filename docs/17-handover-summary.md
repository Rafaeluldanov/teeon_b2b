# 17. Краткая памятка для передачи проекта

## Что сделано

Разработан полноценный B2B-сайт производственной компании TEEON на Next.js 14 + React 18 + TypeScript + App Router + CSS Modules.

### Реализовано:
- 44+ страниц: главная, 8 категорий каталога, 9 методов брендирования, 9 кейсов портфолио, о компании, контакты, сувениры, privacy, requisites
- Полная SEO-инфраструктура: title/description/canonical/OG/JSON-LD/sitemap/robots
- Форма заявки с загрузкой файлов, honeypot, тремя режимами отправки (dev/ethereal/smtp)
- Конфигуратор изделий с мультипозиционной корзиной расчёта
- Защищённая административная часть (login/password через httpOnly-cookie, middleware)
- **Редактор каталога**: категории, модели, варианты, цвета. **Изображение добавляется только к варианту** (не к категории/модели)
- **Редактор портфолио**: coverImage, galleryImages, caseProducts с фото — отображаются на `/portfolio/` и страницах кейсов
- **Редактор контактов**: правки телефона, email, Telegram отображаются в Header и Footer в реальном времени
- Загрузка изображений в S3/MinIO или `public/uploads/` (локально)
- Кнопка «Удалить изображение» в форме варианта каталога и в кейсах портфолио
- E2E-тесты (Playwright, 121 прошло), SEO-аудит, скрипт тестирования формы
- Яндекс.Метрика (подключается через переменную окружения)

### localStorage-ключи

| Ключ | Что хранит |
|---|---|
| `teeon_admin_catalog_models` | Данные каталога (категории → модели → варианты) |
| `teeon_admin_portfolio_cases` | Кейсы портфолио с изображениями |
| `teeon_admin_contacts` | Контакты: телефон, email, Telegram, адрес |
| `teeon_admin_branding_methods` | Методы брендирования |
| `teeon_admin_branding_samples` | Варианты нанесения (образцы) |
| `teeon_admin_home_banner` | Тексты и медиа главного баннера |
| `teeon_admin_page_content` | Тексты страниц |

## Как запустить локально

```bash
# Установить зависимости (один раз)
npm install

# Создать файл переменных окружения
cp .env.example .env.local
# Заполнить .env.local (см. docs/04-environment-variables.md)

# Запустить
npm run dev

# Сайт: http://localhost:3000
# Админка: http://localhost:3000/admin/login/
```

## Что требует следующего этапа

### Высокий приоритет (перед публикацией)
1. Заменить все заглушки в `lib/contacts.ts` на реальные данные
2. Написать реальную политику ПД (`app/privacy/page.tsx`)
3. Добавить реальные фото производства и товаров
4. Настроить production SMTP в `.env.local`
5. Создать Яндекс.Метрику и добавить ID

### Средний приоритет
6. Добавить видео производства (`public/videos/hero-production.mp4`)
7. Встроить карту в страницу контактов
8. Добавить фото в портфолио
9. Создать реальный OG-баннер 1200×630 px

### Долгосрочно (для масштабирования)
10. Подключить CMS/backend для редактирования контента без деплоя
11. Реализовать production S3/Cloudflare R2 для постоянного хранения фото
12. Настроить аналитику конверсий в Яндекс.Метрике (цели: отправка формы)
13. Добавить полноценные логотипы клиентов в секцию «Нам доверяют»
14. Настроить 2FA для администратора
